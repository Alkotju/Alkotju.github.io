ЧТО ТАКОЕ BOOTSTRAP?

BOOTSTRAP (так же известен как Twitter Bootstrap) - свободный набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML и CSS шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейса, включая JavaScript-расширения.

В 2010 году в недрах компании Twitter появился проект Twitter Blueprint. Его задачей было создание системы для разработки новых компонентов внутри компании. В этом же году на Twitter Hack Week проект обрёл первую популярность. Его заметили не только внутри компании, но и за её пределами. Уже в 2011 году проект вышел в открытый доступ под названием Bootstrap.

BOOTSTRAP КАК НАБОР ГОТОВЫХ ЭЛЕМЕНТОВ

Одной из сторон Bootstrap является то, что его можно использовать исключительно как набор уже готовых элементов. Для этого не требуется серьёзных знаний HTML и CSS — достаточно уметь создавать самые простые страницы и прочитать документацию. Например, для создания кнопки используются два класса:

<button class="btn btn-primary">Я — синяя кнопка</button>

Второй подход, который используется в Bootstrap — Atomic CSS. На нём построены утилиты во фреймворке. Atomic CSS — подход, при котором один класс использует одно свойство.

<section class="d-flex bg-white">
  <!-- HTML вёрстка -->
</section>

В этом примере тег <section> получит два свойства: background-color: #fff; и display: flex;.

ПЛЮСЫ ИСПОЛЬЗОВАНИЯ BOOTSTRAP

  • Скорость - Используя готовые компоненты и утилиты Bootstrap, можно быстро ввести новый функционал на сайт.
  • Кроссбраузерность - Компоненты Bootstrap учитывают различия браузеров и оптимизированы таким образом, чтобы минимизировать различия между ними.
  • Адаптивность - Bootstrap имеет готовую 12-колоночную сетку, по которой можно располагать компоненты.
  • Доступность - Разработчики уделяют внимание использованию страниц с компонентами людьми с ограниченными возможностями.
  • Низкий порог вхождения - Создание страницы по компонентам не требует экспертного знания HTML и CSS.

Появление и развитие Bootstrap

Bootstrap — это популярный фреймворк для разработки адаптивных и мобильных веб-проектов, основанный на HTML, CSS и JavaScript. Изначально разработанный как внутренняя библиотека компании Twitter под названием Twitter Blueprint, Bootstrap был выпущен в открытый доступ 19 августа 2011 года. С тех пор он стал одним из наиболее широко используемых инструментов для создания современных веб-интерфейсов.

Одной из ключевых особенностей Bootstrap является его 12-колоночная система сетки, представленная во второй версии фреймворка в 2012 году. Эта система позволяет разработчикам легко создавать сложные макеты, которые автоматически адаптируются к различным размерам экранов и устройствам. Кроме того, Bootstrap предоставляет широкий набор готовых компонентов, таких как кнопки, формы, навигационные панели и карусели, что значительно ускоряет процесс разработки и обеспечивает единообразие дизайна.

С выходом пятой версии Bootstrap разработчики отказались от зависимости от jQuery, улучшили поддержку современных технологий и добавили новые компоненты, что делает фреймворк еще более мощным и гибким инструментом для веб-разработки.

BOOTSTRAP 5 И ПЕРСПЕКТИВЫ РАЗВИТИЯ

Bootstrap продолжает эволюционировать, сохраняя свою позицию как один из самых популярных CSS-фреймворков:

Отказ от jQuery в Bootstrap 5

В Bootstrap 5 полностью переписаны все JS-компоненты с использованием нативного JavaScript:

  • Значительное уменьшение размера фреймворка и повышение производительности
  • Более простая интеграция с современными JavaScript-фреймворками
  • Улучшенная поддержка современных API браузеров

Улучшенная кастомизация через SASS переменные

Bootstrap 5 предлагает более гибкий подход к кастомизации:

$primary: #0074d9;
$border-radius: 0.5rem;
$enable-shadows: true;
$enable-gradients: true;

@import "bootstrap/scss/bootstrap";

Новые компоненты и утилиты

  • Offcanvas — сайдбары, выезжающие с любой стороны экрана
  • Accordion — улучшенный компонент для создания списков с раскрывающимся содержимым
  • Utilities API — система для создания собственных служебных классов
<div class="offcanvas offcanvas-start" tabindex="-1" id="offcanvasExample">
  <div class="offcanvas-header">
    <h5 class="offcanvas-title">Меню</h5>
    <button type="button" class="btn-close" data-bs-dismiss="offcanvas"></button>
  </div>
  <div class="offcanvas-body">
    <p>Содержимое выезжающего меню</p>
  </div>
</div>

Bootstrap-иконки и интеграция с SVG

Официальная библиотека SVG-иконок для Bootstrap с более чем 1,600 иконками:

<i class="bi bi-alarm"></i>
<i class="bi bi-cloud-download"></i>

Web Components и будущее Bootstrap

Bootstrap экспериментирует с Web Components, предлагая компоненты, которые работают на любой платформе без зависимостей:

<bootstrap-card>
  <img slot="image" src="..." alt="...">
  <span slot="title">Заголовок карточки</span>
  <span slot="text">Текст карточки...</span>
  <a slot="button" href="#">Подробнее</a>
</bootstrap-card>

Возможности Bootstrap

Grid система

12-колоночная сетка Bootstrap позволяет создавать отзывчивые макеты, которые адаптируются к различным размерам экрана - от мобильных устройств до широкоформатных мониторов.

Готовые компоненты

Bootstrap предоставляет множество готовых компонентов - навигационные панели, карточки, формы, кнопки, модальные окна и многое другое, что значительно ускоряет процесс разработки.

JavaScript плагины

Встроенные JavaScript плагины добавляют интерактивность компонентам без необходимости писать сложный код: карусели, выпадающие меню, всплывающие подсказки и многое другое.
