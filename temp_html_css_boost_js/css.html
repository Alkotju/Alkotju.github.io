ЧТО ТАКОЕ CSS?

CSS (Каскадные таблицы стилей) позволяет создавать великолепно выглядящие веб-страницы. CSS — это язык для определения того, как документы представляются пользователям — как они оформляются, размещаются и т. д.

В модуле Основы HTML мы рассмотрели, что такое HTML и как он используется для разметки документов. Эти документы будут доступны для чтения в веб-браузере. Заголовки будут выглядеть больше, чем обычный текст, абзацы разбиваются на новую строку и будут иметь пространство между друг другом. То, что вы видите, — это стили браузера по умолчанию. Используя CSS, вы можете до мелочей контролировать, как элементы HTML выглядят в браузере.

СИНТАКСИС CSS

CSS — это язык на основе правил: вы задаёте правила, определяющие группы стилей, которые должны применяться к определённым элементам или группам элементов на вашей веб-странице. Например:

h1 {
  color: red;
  font-size: 5em;
}

Правило открывается с помощью селектора. Этот селектор выбирает HTML-элемент, который мы собираемся стилизовать. Затем у нас есть набор фигурных скобок { }. Внутри них будет один или несколько объявлений, которые принимают форму пары свойства и его значения.

КАК РАБОТАЕТ CSS?

Когда браузер отображает документ, он должен совместить его содержимое с его стилями. Этот процесс идёт в несколько этапов:

  1. Браузер получает HTML-страницу (например, из Интернета)
  2. Преобразует HTML в DOM (Document Object Model)
  3. Затем браузер забирает все ресурсы и описания, связанные с HTML-документом
  4. Браузер анализирует полученный CSS код, сортирует описанные там правила
  5. Дерево представления (render tree) формируется в том порядке, в каком оно затем должно будет отображаться
  6. Затем происходит визуальное отображение контента на странице (этот этап называется «отрисовкой»)

Эволюция и возможности CSS3

CSS3 — это третья версия каскадных таблиц стилей, предназначенная для описания внешнего вида документов, написанных с использованием языков разметки. Первая версия CSS была представлена в 1996 году и включала базовые возможности по стилизации текста, фонов и границ. С течением времени CSS эволюционировал, и в CSS3 были введены значительные улучшения и новые функции.

Одним из ключевых нововведений CSS3 стало модульное построение, позволяющее разрабатывать и обновлять отдельные части спецификации независимо друг от друга. Это ускорило внедрение новых возможностей и улучшило совместимость между различными браузерами. Среди новых функций CSS3 можно выделить поддержку анимаций, переходов, градиентов, теней, а также медиазапросов, которые позволяют создавать адаптивный дизайн, подстраивающийся под различные устройства и размеры экранов.

Благодаря этим улучшениям, CSS3 значительно расширил возможности веб-дизайна, позволяя создавать более динамичные, привлекательные и удобные для пользователя интерфейсы без необходимости использования сложных скриптов или сторонних технологий.

ИННОВАЦИИ И ТРЕНДЫ В CSS

Последние годы CSS развивается особенно быстро, предлагая разработчикам мощные инструменты для создания современных интерфейсов:

Container Queries

В отличие от медиа-запросов, которые основаны на размере экрана, Container Queries позволяют стилизовать элементы на основе размера их родительского контейнера:

@container (min-width: 700px) {
  .card {
    display: grid;
    grid-template-columns: 2fr 1fr;
  }
}

Это революционное изменение в способе создания действительно адаптивных компонентов.

CSS Subgrid

Subgrid позволяет дочерним элементам сетки наследовать и выравниваться по линиям родительской сетки, что значительно упрощает создание сложных макетов:

.parent-grid {
  display: grid;
  grid-template-columns: repeat(9, 1fr);
}

.child-grid {
  grid-column: span 3;
  display: grid;
  grid-template-columns: subgrid;
}

Логические свойства и значения

Вместо свойств, привязанных к физическим направлениям (top, right, bottom, left), логические свойства основаны на направлении текста:

.box {
  margin-block-start: 1em;    /* вместо margin-top */
  margin-inline-end: 1em;     /* вместо margin-right в LTR или margin-left в RTL */
  padding-block-end: 1em;     /* вместо padding-bottom */
  padding-inline-start: 1em;  /* вместо padding-left в LTR или padding-right в RTL */
}

Color функции и пространства

Новые цветовые функции позволяют создавать более гибкие и адаптивные цветовые схемы:

  • color-mix() — для программного смешивания цветов
  • color-contrast() — для автоматического выбора наиболее контрастного цвета
  • Новые цветовые пространства: oklch(), lab(), lch() с лучшим восприятием и гаммой цветов

CSS Nesting

Встроенная поддержка вложенности стилей без использования препроцессоров:

.card {
  background: white;
  
  & .title {
    font-size: 1.5em;
    
    &:hover {
      color: blue;
    }
  }
}

Возможности CSS3

Анимации и переходы

CSS3 позволяет создавать анимации и плавные переходы между состояниями элементов без использования JavaScript или Flash, что делает сайты более интерактивными и привлекательными.

Гибкие макеты

Flexbox и Grid Layout - мощные системы макетов, которые позволяют создавать сложные и адаптивные дизайны с минимальным количеством кода.

Трансформации и эффекты

CSS3 включает возможности для трансформации элементов (вращение, масштабирование), создания теней, градиентов и прозрачности, что существенно расширяет возможности дизайна.
